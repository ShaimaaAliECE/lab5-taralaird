Welcome to the Job Database

To get the categories mentioned in all the jobs and how many times each category was mentioned. type "localhost/categories".

To get all the jobs with a given category, type "localhost/the category name". An example would be: "localhost/java".

To get all jobs in a given city, type "localhost/checkJobsInCity?cityName="the city name". An example would be: "localhost/checkJobsInCity?cityName=Ottawa"
